  //  bg_mat_position ({text{;text}})  // $1: name of a form object  - default is OBJECT Get name(Object current)  // $2: name of the bg object  - default is 'bg_mat'  // Written by: Kirk as Designer, Created: 02/14/19, 13:30:13  // ------------------  // Purpose: 1) position the bg_mat under the named object  //          2) make it visible  // BACKGROUND MAT  // The 'bg_mat' is an object on the form. It should be on Page 0 and be   // positioned on the 'floor'. The simplest thing to do is make it a rectangle object. Fill with  // the color you prefer. It could be a picture variable as well.  // Whatver type of object it should have these properties:   //  hidden by default  //  hide focus rect  //  no border  //  transparent  // It can be places any where and made very small.   // TO USE IT  // The object you want to highlight must have the On getting foucs and On loosing focus   // events enabled.   // Place this method in the the On getting focus code.   // Use bg_mat_hide in the On loosing foucs code.C_TEXT($1;$object)C_TEXT($2;$bgObj)C_LONGINT($x;$y;$x1;$y1)$object:=OBJECT Get name(Object current)$bgObj:="bg_mat"If (Count parameters>0)$object:=$1If (Count parameters>1)$bgObj:=$2End if End if OBJECT GET COORDINATES(*;$object;$x;$y;$x1;$y1)OBJECT SET VISIBLE(*;$bgObj;True)OBJECT SET COORDINATES(*;$bgObj;$x-2;$y-2;$x1+2;$y1+1)
